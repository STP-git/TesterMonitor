About

Tester Monitoring System

Version: 1.0.0

A lightweight real-time monitoring dashboard for tracking tester status by scraping data from multiple tester URLs.

Features:

  • Real-time monitoring with Server-Sent Events
  • Configurable dashboard layout
  • Tester management through UI
  • Auto-refresh with configurable intervals
  • Responsive design

Technology Stack:

  • Backend: Bun with Alpine Linux
  • Frontend: Vanilla HTML/CSS/JavaScript
  • Web Scraping: Axios + Cheerio
  • Real-time: Server-Sent Events
  • Storage: JSON configuration
